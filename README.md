## Function
- Writing(CURD)
- List-Paging(infinite scrolling)
- Maintain login session
- Search keyword
- Custom calendar(diary)
- Like & Comment
- Upload background and profile thumbnail images
- Personal information encryption: SHA2(passwrod), AES(phone, email)
